# Tux BattleGround Game Project

![ ](https://github.com/AlexTutberidze/sfml-game/blob/master/design/WARRIORTUX.png?raw=true)

## Build:

```bash
cd build
./build.py
```


## Run:

Open terminals and execute both files separately.

first run server.exe
then run client.exe

```bash
cd ../bin
./server.exe
./client.exe
```
